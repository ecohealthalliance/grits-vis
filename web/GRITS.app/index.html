GRITS visualization application demo containing four panels
  1. A geojs mapping window using an openstreet maps base layer rendered with webgl and a d3 layer for markers and user interaction.
  2. A d3 based force layout that arranges data by custom data fields.
  3. A vega time-line visualization showing the number of selected incidences as a function of time.
  4. A tangelo dendrogram plugin that diagnoses diseases based symptoms.
Warning: The symptoms data is generated randomly.
